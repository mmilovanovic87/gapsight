# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the GapSight project. ADRs document the reasoning behind significant technical decisions so future contributors understand **why**, not just **what**.

| ADR | Decision | Status |
|-----|----------|--------|
| [001](001-commonjs-for-gapsight-core.md) | Use CommonJS for gapsight-core | Accepted |
| [002](002-dual-constants-files.md) | Maintain dual constants files (browser ESM + Node.js CJS) | Accepted |
| [003](003-localstorage-no-auth.md) | Use localStorage with no authentication | Accepted |
| [004](004-vercel-deployment.md) | Deploy on Vercel free tier | Accepted |
